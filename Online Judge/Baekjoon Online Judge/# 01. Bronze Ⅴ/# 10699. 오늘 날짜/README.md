# BOJ 10699 : 오늘 날짜
- 난이도 : Bronze 5
- 오늘 날짜 출력
- 문제 : [링크](https://www.acmicpc.net/problem/10699)

---

## 풀이
```python
print('2022-09-21')
```
- 채점 서버의 시간대는 utc+0이고, 한국은 utc+9이다.
- 채점 서버의 시간대는 한국 시간에서 9시간을 뺀 값이다.
- 내가 푼 시점은 9월 21일 19시이므로, 채점 시간대는 9월 21일 10시이다.
- 따라서 2022년 9월 21일을 출력하면 되는 것이다.
