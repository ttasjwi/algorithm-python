# BOJ 02754 : 학점계산

- 난이도 : Bronze 5
- 등급 입력 받고 대응되는 점수 반환하기
- 문제 : [링크](https://www.acmicpc.net/problem/2754)

---  

## 풀이
```python
scores = {
    'A+': 4.3,
    'A0': 4.0,
    'A-': 3.7,
    'B+': 3.3,
    'B0': 3.0,
    'B-': 2.7,
    'C+': 2.3,
    'C0': 2.0,
    'C-': 1.7,
    'D+': 1.3,
    'D0': 1.0,
    'D-': 0.7,
    'F': 0.0
}
grade = input()
print(scores[grade], end='')

```
- 딕셔너리를 통해 접근하면, 해시테이블 자료구조이므로 O(1)로 데이터에 접근할 수 있다.
