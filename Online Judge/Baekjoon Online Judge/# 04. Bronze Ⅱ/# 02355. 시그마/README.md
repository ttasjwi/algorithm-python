# BOJ 02355 : 시그마
- 난이도 : Bronze 2
- 범위 내의 연속된 숫자 합 구하기
- 문제 : [링크](https://www.acmicpc.net/problem/2355)

---  

## 풀이
```python
a, b = map(int, input().split())
answer = ((a + b) * (abs(b - a) + 1)) // 2
print(answer, end='')

```
- 연속된 범위의 수의 합은, 첫항의 값, 끝항의 값을 합하고, 여기에 항의 갯수를 곱한뒤 2로 나누면 된다.
- 근데 여기서 a,b의 대소관계가 명확하지 않다. 따라서 양쪽을 고려해야한다. 여기서는 절댓값 처리를 하여 두 수의 차를 기반으로 항의 갯수를 구했다.
- 계산 후 출력
